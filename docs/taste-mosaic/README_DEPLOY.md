# Taste Mosaic 배포 안내

Google에서 `Taste Mosaic`을 검색해 들어오게 하려면 먼저 공개 URL이 필요합니다.

## 가장 쉬운 배포 방법

1. `taste-mosaic` 폴더를 GitHub 저장소에 올립니다.
2. GitHub Pages, Netlify, Vercel 중 하나로 정적 사이트 배포를 켭니다.
3. 공개 URL이 생기면 Google Search Console에 등록합니다.
4. URL 검사에서 `index.html` 주소를 제출하고 색인을 요청합니다.

## 포함된 검색 준비

- 각 페이지의 title과 description
- Open Graph 공유 메타
- `robots.txt`

배포 후 Google에 바로 뜨는 것은 아니고, 보통 색인까지 시간이 걸립니다.
